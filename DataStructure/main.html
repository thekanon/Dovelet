Abstract Data Structure
  • list 추상적 자료형, 자료구조의 하나. 순열(Sequence)이라고도 불리며, 순서를 가지고 일렬로 나열한 원소들의 모임으로 정의한다
  • Stack 후입선출(Last In First Out—LIFO) 특성을 가지는 자료구조(Data Structure)를 일컫는다.
  • Queue 선입선출(First In First Out; FIFO)의 자료구조. 대기열이라고도 한다.
  • Tree 부모-자식 관계로 정의하고, 부모에서 자식으로 간선이 이어져 있는 유향 그래프이다.
  • Heap 힙은 특정한 규칙을 가지는 트리로, 힙을 이용해서 우선순위 큐를 구현할 수 있다.
    sort
  • Bubble sort Bubble sorting is the simplest arrangement.
  • Heap sort Hip sorting does not require any additional memory, and always has the advantage of performing O(nlognn).
  • Quick sort It represents the best performance in an average situation.
